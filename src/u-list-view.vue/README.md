# ListView

## Examples
### Basic

``` html
<u-list-view>
    <u-list-view-item>Apple</u-list-view-item>
    <u-list-view-item>Banana</u-list-view-item>
    <u-list-view-item>Orange</u-list-view-item>
</u-list-view>
```

### `value` & `selected`

``` html
<u-list-view>
    <u-list-view-item>Apple</u-list-view-item>
    <u-list-view-item>Banana</u-list-view-item>
    <u-list-view-item selected>Orange</u-list-view-item>
</u-list-view>
```
